# Clustring Algorithmes From Scratch

This repository contains **from-scratch implementations of clustering algorithms** in Python, built for learning, experimentation, and comparison.  
All algorithms are implemented without relying on high-level machine learning libraries, enabling a deeper understanding of fundamental mechanics, objective functions, cluster behavior, and limitations.

Clustering is a core unsupervised learning technique used in:
- exploratory data analysis
- segmentation and pattern discovery
- anomaly detection
- image / spatial grouping
- high-dimensional data summarization
